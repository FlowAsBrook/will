## 指令

- 安装tar.gz，解压之后，执行

  ```shell
  python3 setup.py install
  ```

  

- 查看指定依赖版本

  ```shell
  pip3 show package_name
  ```

  

## 信创

- 安装python3-devel

  ```shell
  dnf install python3-devel
  ```
