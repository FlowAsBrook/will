# ISSUES

## search

`searchKeyWord is:issue is:closed repo:Alamofire/Alamofire` 

这条搜索，searchKeyWord是搜索关键字, `is:issue` 表示我们要搜索 issue， `is:closed` 表示已经关闭的 issue， `repo:Alamofire/Alamofire` 表示我们只搜索这个仓库范围的 issue
