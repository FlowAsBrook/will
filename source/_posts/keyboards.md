# niz plum

```
对照
ctrl -> control
win/lock -> option
alt -> command
```

## 左 Fn

- 左Fn + num

  ```shell
  Fn + 1 = 调低亮度
  Fn + 2 = 调高亮度
  Fn + 3 = 调出所有应用窗口
  Fn + 4 = spotlight
  Fn + 5 = voice
  Fn + 6 = F6
  Fn + 7 = 播放后退
  Fn + 8 = 播放暂停/恢复
  Fn + 9 = 播放前进
  Fn + 10 = 静音
  Fn + 11 = 音量减小
  Fn + 12 = 音量增大
  ```

  

## 右 Fn

- 锁定键盘

  ```
  右Fn + Esc
  ```

- 电源与蓝牙切换

  ```
  右Fn + F11键
  ```
